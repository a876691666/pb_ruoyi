# 快速开始

## 简介

PocketBase RuoYi 是一个基于 PocketBase 后端和 Vben Admin 前端的全栈开发框架，旨在帮助开发者快速构建现代化的企业级应用。

## 特性

- **🚀 快速开发**：开箱即用的完整解决方案
- **⚡ 高性能**：基于 Go 的 PocketBase 后端，性能卓越
- **🔐 安全可靠**：内置完整的认证授权系统
- **📦 轻量部署**：单文件部署，简单高效
- **🎨 现代化 UI**：基于 Vue 3 + TypeScript 的现代前端

## 技术栈

### 后端
- [PocketBase](https://pocketbase.io/) - 开源后端框架
- SQLite - 轻量级数据库
- Go - 高性能编程语言

### 前端
- [Vben Admin](https://github.com/vbenjs/vue-vben-admin) - Vue 3 企业级管理系统
- Vue 3 - 渐进式 JavaScript 框架
- TypeScript - JavaScript 的超集
- Vite - 下一代前端构建工具

## 安装

### 环境要求

- Node.js 18+ 
- Go 1.21+
- pnpm 8+
